Ratelations

The third project of my coding bootcamp at University of California Irvine. The idea behind this project is to eliminate communication between couples. You login to your account and add a partner into a relationship with you. As days go by you log your daily progress. If you feel the day went great then you rate it 5 stars, if the day was bad then rate 1 star. The chart keeps track of the days and if you have multiple bad days for a month straight you'll get close to the breakup point. We added a comment section so that you can add friends to your relationship. they can comment on yours and the significant others post.

PokeBattles

The second project of my coding bootcamp at University of California Irvine. With this project we used a Pokemon api and database. The idea behind the project is to pick the best team to fight against your enemies team. So to do that you can enter a pokemon's name or use the random name generator we built into the program. After inputting a name it will generate a list of the best 5 Pokemon to fight against the one entered. We also made it so that you can store your teams so that the next time you login, you will see all the previous teams generated.
